## Overview
**Wallpaper Engine** allows you to set beautiful animated, interactive, and video live wallpapers on your Windows desktop.

Supports high-quality 4K/8K wallpapers, Workshop content, audio reactivity, and much more.

> [!Important]
> This is a free community version.

## Key Features
- Thousands of animated & interactive wallpapers
- Full Steam Workshop support
- 4K, 8K and Ultra HD resolution
- Low CPU & GPU usage
- Audio visualizer wallpapers
- Custom scene creation
- Multiple monitors support
- Hotkey & auto pause on fullscreen
- Razer Chroma & Corsair iCUE integration

## How to Use
1. Download the latest release
2. Extract the archive
3. Run `WallpaperEngine.exe` as Administrator
4. Choose and apply your favorite live wallpaper
5. Enjoy your animated desktop!
